# Sky Ball 🎮

👉 Play the game: https://crystal-x-blade.itch.io/sky-ball

---

## 💭 About the Game

**Sky Ball** is a 3D balance and movement game where you control a rolling ball across narrow sky platforms filled with obstacles.

The goal is simple:

> Reach the end without falling.

---

## ✨ Features

- 🎮 Smooth third-person movement  
- ⚙️ Physics-based gameplay  
- 🧩 Moving platforms and obstacles  
- 🎯 Precision-based challenge  
- 🌌 Minimal sky environment  

---

## 🎮 Controls

| Action | Key |
|--------|-----|
| Move   | WASD |
| Jump   | Space |
| Camera | Mouse |
| Pause  | ESC |

---

## 🛠️ Built With

- Unity Engine  
- C#  

---

## 🚀 How to Play
```
1. Download the game from itch.io  
2. Extract the ZIP file  
3. Run `SkyBall.exe`

```
⚠️ Windows might act a little dramatic and say “this app looks suspicious” 😤  
Don’t worry — it’s just because the game is new.

👉 Click **More info → Run anyway** and enjoy!
